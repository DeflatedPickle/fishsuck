# fishsuck
It's a physics engine

> Though this engine is written to be 3D, all the supplied renderer's are 2D

## Deconstruction

```
+ + + + + + + + + +
+                 +
+  @ @ @ @ @ @ @  +
+  @   $ $ $ $ @  +
+  @ $ $     $ @  +
+  @ $   $ $ $ @  +
+  @ $ $ $     @  +
+  @ @ @ @ @ @ @  +
+                 +
+ + + + + + + + + +
```

`+` = Space

`@` = Body

`$` = Shape/s
